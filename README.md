# frontplate-cli

This CLI is beta version.

## Usage

```
npm i frontplate-cli -g
```

## Command

```
frp create <APP_NAME>
frp build
frp serve
```

### Options

```
-p, --production
```

## Speed up

frontplate: 15s
default-grayhounds: 14s (※ 画像のminify含む)
frontplate-cli: 9s (※ 画像のminify含む)
